The **Deep Research Agent Backend** is a Python-based multi-agent orchestration system designed for deep research workflows using LLMs, LangGraph, and various automation agents.  
This backend serves as the core engine responsible for query analysis, agent execution, and delivering structured insights.

---

##  Key Capabilities

- Multi-agent orchestration using **LangGraph**
- Intent parsing & query understanding
- Specialized research agents:
  - Search Agent
  - Web Crawl Agent
  - Summarization Agent
  - Analysis Agent
  - Report Generator
- FastAPI-powered REST API
- Modular, plug-and-play Python architecture
- Support for OpenAI, Gemini, Anthropic, Groq LLMs
